How to run in other computers:
0. Copy/clone this directory/repo to other computers

1. Set the current path as the INSTALL variable in the scripts:
    javac-algs4 and java-algs4

    Then enter this directory as working directory

2. Compile
    eg: 
         ./javac-algs4 src/main/java/edu/princeton/cs/algs4/ST.java src/main/java/edu/princeton/cs/algs4/FrequencyCounter.java 
3. Run
    View the run.cmd, eg:
        ./java-algs4 edu.princeton.cs.algs4.FrequencyCounter 10 < /ExtDisk2/github/algs4/algs4-data/algs4-data/leipzig1M.txt
